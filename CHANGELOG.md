# Changelog

## Initial Release

- Laravel/PHP fullstack alias manager for Bash and Zsh.
- Safe installer that keeps existing aliases, Daily Favorites, and config by default.
- Checkbox setup UI with `dialog` or `whiptail`, plus a text fallback.
- Searchable alias registry with add, change, remove, list, and help commands.
- Daily Favorites workflow for frequently used aliases.
- Alias packs for Laravel, Sail, Composer, npm/Vite, Docker, Git, quality tools, security checks, and project workflows.
