# Help Topics

Use the built-in help center after install:

```bash
dam help
dam help topics
dam help quick
dam help linux
dam help git
dam help github
dam help docker
dam help php
dam help frontend
dam help laravel
dam help sail
dam help quality
dam help security
dam help workflow
dam help daily
dam help config
dam help add
dam search WORD
dam help alias NAME
```

High-value discovery commands:

```bash
dam search sail
dam help sail
dam help git
dam help frontend
dam list
```

Sail users can install the Sail pack and use either readable aliases or short aliases:

```bash
dam preset sail
sail artisan migrate
sailup
sailshell
sup
smig
```
