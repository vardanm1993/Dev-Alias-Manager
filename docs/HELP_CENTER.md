# Help Center

The main entry point:

```bash
dam help
```

Useful discovery commands:

```bash
dam help topics
dam search WORD
dam list
dam help alias NAME
dam help quick
```

Examples:

```bash
dam search route
dam search docker
dam search sail
dam help alias myroutes
dam help alias qa
dam help alias sail
```

Common first installs:

```bash
dam preset fullstack
dam preset sail
dam daily choose
```

Sail aliases include both readable names such as `sail`, `sailup`, and `sailshell`, plus short daily names such as `sup`, `sdown`, `sart`, and `smig`.
